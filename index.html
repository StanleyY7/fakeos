Fake OSX Yosemite

Welcome to this project, my own personal website mock-up of the Mac OSX (version 10.10) Yosemite operating system. This project was made entirely using HTML, CSS, SCSS and Javascript.


All apps do a particular action as well. If you are interested as to how I created this project, check out my Github repository by using the Github app available at the bottom dock.


Alternatively if you wanted to check out my other projects, select the :S app from the dock.
